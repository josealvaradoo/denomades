# Denomades Next.js App

It's a next.js project with a structure to use redux (redux ducks) and redux-saga. Also, use the service concept to make api calls

### Install
```
npm install
```

### Enviroment vars
Duplicate the `.env.example` file and name it `.env`

### Run project on development mode
```
npm run dev
```

### Run eslint
```
npm run lint
```
